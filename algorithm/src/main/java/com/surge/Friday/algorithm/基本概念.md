    T(n) :相对执行时间函数（基本操作执行次数）。
    f(n)：存在一个函数fn 当n趋近于无穷大的时候，Tn/fn为常数级，则fn是Tn的常数级。
    记做：T(n) = O(f(n))，O为算法的渐进时间复杂度，简称为时间复杂度。
